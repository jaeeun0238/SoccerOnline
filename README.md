# 💡 시작 가이드
###
<h3>Requirements</h3>
For building and running the application you need:
 
 - Node.js 18.x
 - Npm 9.2.0
 
<h3>Installation</h3>

```
$ git clone https://github.com/jaeeun0238/SoccerOnline.git
$ cd SoccerOnline
```
#### Run Method
```
$ npm install yarn
$ yarn
$ node app.js
```
